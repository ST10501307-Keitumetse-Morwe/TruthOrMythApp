# TruthOrMythApp

# Hello World

Welcome to my 'Truth Or Myth' application which has a set of 9 questions about different things and you have to decide which is a myth and which is the truth. I created it in order to help people differntiate between something that is true and something that isn't true. Hopefully, this application help people in the real world.

On my Review Screen in the application , I initially had a textview to display all the questions along with the correct answer and appropriate feedback but my text was too long to fit into the textview so I ended up using a Scrollview from the pallete and put my textview inside it so that you could see everything by scrolling up.
